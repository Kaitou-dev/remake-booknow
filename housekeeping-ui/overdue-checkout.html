Handling Overdue Checkouts

1. Inspect Room

Visit the room to check if the guest is still present or has left without checking out.

2. Contact Guest

If the guest is reachable, remind them to complete checkout via the app.

3. Force Checkout

If the guest has departed without checkout, staff can force checkout to release the room.

4. Create Cleaning Task

After checkout, a cleaning task will be automatically created for the room.
